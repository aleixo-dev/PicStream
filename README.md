# PicStream
PicStrem é um aplicativo para baixar qualquer tipo ou categoria de foto em alta resolução em apenas alguns cliques.

## 🏗️ Build With
- 100% Kotlin
- Jetpack Compose
- Paging3
- Retrofit2
- OkHttp
- Download Manager
- Broadcast Receivers
- Alarm Manager
- Koin
- Room
- Connectivity Manager
- e mais...
